# What is it?

A simple script to encrypt a folder containing pdf files using `qpdf`.

# Pre-requisite

## macOS
```
brew update
brew install qpdf
```

# How to use it?
```
$ ./bin/enc_pdir.sh input_pdf
Enter Password:
encrypting input_pdf/hello1.pdf ...
encrypting input_pdf/hello2.pdf ...
encrypting done
```
